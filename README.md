# Meli test

### Instalacion de pendencias

```bash
$ yarn deps
```

### Inicio server

```bash
$ yarn start
```

### Inicio server (dev mode)

```bash
$ yarn dev
```

### Servir la app

Build primero:
```bash
$ yarn build:client
```

Y ahora si se puede levantar el client (en una terminal aparte)
```bash
$ yarn server:client
```
